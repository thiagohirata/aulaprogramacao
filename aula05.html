Aula 5 - Arrays e Objetos

Para que servem Arrays?

var pessoa1 = 'Ana';
var pessoa2 = 'Luis';
var pessoa3 = 'Juliano';
var pessoa4 = 'Artur';
notificar(pessoa1);
notificar(pessoa2);
notificar(pessoa3);
notificar(pessoa4);

Um objeto Array é usado para armazenar múltiplos valores em uma única variável

var array1 = [ 'Ana', 'Luis', 'Juliano', 'Artur' ];
notificar(array1[0]); //o número em colchetes
notificar(array1[1]); //indica a posição
notificar(array1[2]); //iniciando em '0'
notificar(array1[3]);

Loops + Arrays

var array1 = [ 'Ana', 'Luis', 'Juliano', 'Artur' ];
var x = array1.length; //length é uma propriedade que retorna a quantidade de valores
for(var i = 0; i < x; i++) {
	notificar(array1[i]);
}

Manipulando o conteúdo de Arrays

var array1 = []; // array vazio
array1[0] = 123; // no javascript, o array muda de tamanho automaticamente
array1[1] = 'Elemento 2';
array1[10] = 'Abacaxi'; //agora existem buracos no array!
array1.push('Abacaxi'); //acrescenta um elemento ao fim do array

Objetos (em Javascript)

var pontosJogador1 = 1;
var pontosJogador2 = 2;
var posicaoXJogador1 = 0;
var posicaoXJogador2 = 0;
var posicaoYJogador1 = 100;
var posicaoYJogador2 = 100;

Um objeto permite guardar diversos valores em uma variável, mas cada valor é acessível a partir de um nome (string), e não um índice (número).

var jogador1 = {
	pontos: 0,
	x: 0,
	y: 0
};
var jogador2 = {
	pontos: 0,
	x: 100,
	y: 100
};
